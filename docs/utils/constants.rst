constants.hpp
=============

Contains various constants.

.. c:macro:: PLANCK_SI

    Planck's constant in :math:`\mathrm{Js}`.

.. c:macro:: LIGHT_SI

    The speed of light in :math:`\frac{\mathrm{m}}{\mathrm{s}}`.

.. c:macro:: AVOGADRO

    Avogadro's number in :math:`\mathrm{mol}^{-1}`

.. c:macro:: AU_TO_KG

    Conversion factor from atomic mass units to kilograms.

.. c:macro:: BOHR_TO_M

    Conversion factor from Bohr radii to meters.
